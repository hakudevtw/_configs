# Applications

Apps to install when setting up a new Mac environment. See each project's documentation for installation instructions.

## Must Have

- **[Cursor](https://cursor.com)** — AI-powered code editor built on VS Code, with integrated coding agents and chat.
- **[Claude](https://claude.com/download)** — Desktop app for Claude chat, Cowork, and Claude Code. Syncs across desktop, web, and mobile.
- **[OrbStack](https://orbstack.dev)** — Fast, lightweight Docker and Linux containers for macOS.
- **[Git](https://git-scm.com/download/mac)** — Distributed version control system for tracking code changes.
- **[Ghostty](https://ghostty.org)** — Fast, native terminal emulator with GPU rendering and modern features.
- **[Raycast](https://raycast.com)** — Productivity launcher for macOS — app search, shortcuts, snippets, and extensions.
- **[Scroll Reverser](https://pilotmoon.com/scrollreverser/)** — Per-device scroll direction settings. Useful when mixing a mouse and trackpad with different scroll preferences.
- **[Karabiner-EventViewer](https://karabiner-elements.pqrs.org/docs/manual/misc/eventviewer/)** — Utility for inspecting keyboard events. Helpful when writing Karabiner-Elements rules.
- **[Karabiner-Elements](https://karabiner-elements.pqrs.org/)** — Keyboard customization for macOS — remap keys, create complex shortcuts, and tune input devices.
- **[Google Chrome](https://www.google.com/chrome/)** — Web browser for development and daily browsing.
- **[Notion](https://www.notion.com)** — All-in-one workspace for notes, docs, tasks, and wikis.
- **[Notion Calendar](https://www.notion.com/product/calendar)** — Calendar app for scheduling and time blocking. Syncs with Google Calendar and works alongside Notion.
- **[LINE](https://line.me)** — Messaging app for chat and calls.
- **[Discord](https://discord.com/download)** — Voice, video, and text chat for communities and teams.
- **[Bitwarden](https://bitwarden.com/download/)** — Open-source password manager for storing and syncing credentials.
- **[NordVPN](https://nordvpn.com/download/)** — VPN for encrypted browsing and secure network access.
- **[IINA](https://iina.io)** — Modern macOS video player built on mpv, with a clean native interface.
- **[Gemini](https://gemini.google.com/app)** — Google's AI assistant app for chat, search, and productivity.

## Optional

- **[Google Antigravity](https://antigravity.google)** — Agentic IDE from Google for AI-assisted development. Optional alternative or complement to Cursor.
- **[Visual Studio Code](https://code.visualstudio.com/)** — Lightweight code editor from Microsoft. Useful as a secondary editor or for extensions Cursor doesn't support.
- **[Caffeine](https://www.caffeine-app.net/)** — Menu bar utility that prevents your Mac from sleeping while active.
- **[Zappy](https://zapier.com/zappy)** — Fast screenshot, GIF, and screen recording tool with annotations.
